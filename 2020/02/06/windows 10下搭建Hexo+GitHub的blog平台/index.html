    环境搭建

    安装Git

    安装node.js

    安装 Hexo

    $ npm install -g hexo-cli

    出现以下信息即为安装成功。

    $ hexo -v

    初始化blog

    选择一个存放blog的位置，然后在此位置下初始化。

    $ hexo init myblog
$ cd myblog

    会在当前文件夹下产生一个myblog的文件夹。

    $ npm install

    会在myblog目录中安装node_modules。

    $ hexo s #或者hexo server

    在浏览器中输入localhost:4000就可以访问刚刚创建好的blog了。
